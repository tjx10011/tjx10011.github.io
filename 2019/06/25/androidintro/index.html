Android玩机经验分享（含相关推荐及避坑指南）

发表于 2019-06-25 更新于 2019-06-26 分类于 技术 ， Android

写在前面

本文主要总结了博主近段时间有关安卓解锁、刷机、个性化使用的经验。使用的设备是小米8（6+128），文中会涉及对一些第三方ROM的简单评价，Magisk模块介绍，以及问题解决方案。希望能够让更多人通过更容易的方式享受玩机的乐趣。本文按照一般操作步骤顺序展开。

有关解锁Bootloader

如果要刷第三方ROM或者root（使用Magisk或者Xposed）的需求，就必须要先解锁Bootloader。手机能否解锁依据厂商而定。目前大致的信息如下：

  1. 小米：自用设备。目前政策是使用90天后可用小米帐号进行解锁，但是具体计时方法比较玄学，总之是需要先忍着用一段时间的ADUI。
  2. 华为：全系列不能解锁。甚至不能使用第三方启动器。（没错博主是华为黑）
  3. 一加：可以解锁，较为方便。
  4. 诺基亚：依照机型情况不同。光卡提供这方面的服务。
  5. 其他国内厂商依照机型情况不同，具体可搜索引擎。

刷入TWRP

TWRP是通用的Revocery，在刷机等操作过程中是必要的。因此一般解锁后先需要刷入TWRP。可以在XDA中搜索自己的机型进入相应论坛，在其中可以找到相应的TWRP刷机包。具体操作过程网上教程很多。

第三方ROM

在XDA中可以找到自身机型流行的第三方ROM，并有相应的安装指导。下面介绍几种主流的第三方ROM，都是基于AOSP开发的。

Havoc OS

Sourceforge Github

博主使用的第一个第三方ROM。特点是功能多，非常多，而且具有很强的可定制性。打开设置页后有一整页该ROM专属的设置列表，可以对通知栏，快速设置，系统UI，息屏显示，最近任务，导航栏，手势（自带了多种全面屏手势方案），省电控制，游戏模式等方面进行个性化。随便找了几张图放在下面。虽然它确实提供了很多独特的功能体验，但也具有BUG多的明显缺点。因此在某次严重BUG之后我换用了其他的ROM。如果确实对定制性和功能有较大需求的可以推荐。

Pixel Experience

官方网站

一个模仿Pixel系列使用体验的第三方ROM。在原生Android系统之上加入了Pixel系列的特殊功能（例如与Google系列服务的良好交互），内置了GApps，因此可以方便地直接使用谷歌相机，助手等功能。甚至谷歌认为我买了一台新Pixel。图一是LiveDisplay对屏幕进行调色的功能。除了这些之外没有什么额外的功能。BUG有，但比较少见也不严重。是博主目前使用的ROM，个人比较喜欢。

LineageOS

官方网站

老牌第三方ROM，上面的Havoc等很多ROM是以它为基础开发的。特点是简介清爽，对于低配机型比较友好。没有额外功能，体验比较基础。如果比较喜欢简洁的系统可以选择。

Magisk模块推荐

在刷入TWRP后可以在TWRP中刷入Magisk，安装Magisk Manager，即可称Root了设备，也就可以安装以下的模块。

Greenify

XDA

让“绿色守护”获得完整的功能，充分限制各种流氓应用占用后台。

UniversalGMSDoze

XDA

一般刷第三方ROM后都会刷Gapps，但是谷歌应用后台同步消耗大量电量，尤其在国内网络条件下。该模块可以解决这一问题。

ACC(Advanced Charging Controller)

XDA Github App

通过控制充电过程减少电池损耗。安装App可以进行设置，例如开始充电和停止充电的电量比例，温度控制，电压控制（部分机型）。

Riru

Github

Riru有一个核心模块，安装后可选择安装其他的功能模块。例如EnableLocationReport模块可以开启谷歌的位置记录服务，进而启用负一屏Discover功能，谷歌地图Timeline，Google Fit等功能。（安装后需要清除谷歌系软件的数据并拔卡启动一次）StorageRedirect模块则配合存储重定向软件使用，让手机存储空间不再乱七八糟。

前车之鉴

由于博主刚开始折腾的时候经验不足，导致手机变砖数次，造成了极大的损失。希望大家刷机都能避免翻车。

有关备份

备份不规范，变砖两行泪！

参考资料：link

对于玩机用户来说，比较推荐的是钛备份和Swift Backup。钛备份个人常用，功能确实强大，UI确实让人头大。听说Swift Backup近日更新了版本，可以尝试。另外还可以使用TWRP备份整个分区，方便出问题后直接恢复。如果手机内有重要数据，在任何刷机，安装Magisk模块等操作前请备份！

Firmware?Vendor.img？

第三方ROM包中一般不包含Firmware，需要手动升级。有时候升级Firmware可以解决一些奇怪的问题。小米系列可以通过该网站下载固件包，在TWRP刷入即可。

在刷第三方ROM时有时会出现提示Vendor.img版本不够不能安装的情况，这时需要手动刷入。方法是下载官方系统的最新线刷Fastboot包，解压找到Vemdor.img文件，在TWRP内刷到Vendor分区。

如何救砖

这里救砖指的是刷机失败只能进入Fastboot模式的情况，如果连Fastboot都进入不了，情况就比较严重，可能需要寻找厂商解决。

一般的救砖流程：下载官方系统Fastboot线刷包利用官方指示的工具刷入（此时应该会清除全部数据），确认手机能够正常工作后重复刷TWRP——刷第三方ROM——刷Magisk，GApps——恢复数据的步骤。
