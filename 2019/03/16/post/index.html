9102年的初心者科学上网指南（三）

发表于 2019-03-16 分类于 技术 ， 网络

服务端配置

下载安装

使用bash <(curl -L -s https://install.direct/go.sh)安装。
随后使用cd /etc/v2ray/切换到配置文件文件夹。
使用nano config.json进行编辑。

配置文件

配置文件的各项设置十分丰富，在官方网站，白话文教程中有详细介绍，在此不作重复。

在此分享个人使用的配置文件基本格式。采用无混淆的一般传输方式，配置较为简便且不易出错。下面是一个示例的双端口6用户配置文件。

有几点注意事项。

  • 端口记得填写安装bbr时选择使用bbr加速的端口(如安装bbr时进行过选择端口步骤)。
  • 注意逗号的语法规则。在多项并列之间需要逗号，最后一项则不用。注意uuid6的}后没有逗号。
  • 防止低级错误，例如英文符号打成中文，括号不成对等。
  • Uuid用这个网站生成。
  • 可以先在本地用Atom/sublime等编辑好，再复制到VPS，本地保存副本，以便维护。

启动

编写完配置保存后，使用systemctl restart v2ray重启服务。
使用systemctl status v2ray.service查看服务状态。如果结果为绿色active(running)则为正常，否则会提示错误信息。到这一步，服务端配置完成，可以断开SSH连接了。

客户端配置(PC)

客户端安装

我们选择Clash for windows。因为这一客户端配置简便，功能丰富，(最重要的)界面好看。页面中也提供了macOS版本的链接。
访问Releases页面下载最新exe安装包。安装后在任务栏找到其图标，右键选择Dashboard进入界面。
在此页面可以了解详细使用方法。

配置文件

该客户端配置文件采用Yml编写。在此页面有较为详细的编写示例。同样，分享个人使用的配置文件示例。

多条线路直接在Proxy下按示例添加多行，再将名字填在Proxy Group下的proxies即可。最后的Rule规则部分设置了对google和onedrive通过代理，国内网站直连的分流策略。
同样，也有一些注意事项。

  • NAT类型需要填写映射到的外网端口。
  • 避免低级错误，注意每行-的后面有一个空格再接内容。
  • 本地做好备份，多存几个副本。配置文件拖进之后会在文件夹中消失。

启动

在客户端中选择左侧的Profiles选项。将配置文件拖进窗口，点击出现的所需配置文件，变成橙色即为正在使用。切换到Proxies选项卡查看线路情况，有测试延迟功能(不太准确)。

测速

到这里，你已经可以在PC上进行科学上网。你会想了解速度和网络情况。以下工具：

  • Speedtest选择单线程较为准确。
  • Youtube搜索4K，点进去选择清晰度后在播放界面右键，点击显示详细信息，即可查看网速。
  • Ip检测可检测目前使用的IP数据。(新工具，感谢开发的大佬。)

移动端配置(安卓)

客户端安装

我们选择BifrostV。在apkpure或谷歌商店搜索下载。

配置

右上角点击加号选择手动设置。输入IP，端口等信息。“用户ID”为UUID。Alterid填写64。打开Mux。路由选择“绕过局域网与中国大陆地址”。远程DNS填写8.8.8.8。配置完成后连接即可。
可以生成链接或二维码进行共享。

关于DNS的一点说明

使用中发现，对于某些VPS，如上的设置不能成功，chrome错误代码提示DNS相关。可以尝试打开“使用内置DNS”并尝试不同的DNS设置。本人才疏学浅，不能弄清背后的原因。欢迎在评论区分享。

到这里，你已经完全掌握了自建v2ray科学上网的全部流程，迈出了走向自由互联网的重要一步。恭喜！
当然，v2ray和VPS的应用不仅于此。本系列下一篇将讲述v2ray其它的应用，例如tg代理等等。
